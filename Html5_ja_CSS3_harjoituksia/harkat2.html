T24 tee Pre-elementti ja code elementti sen sisään

Pre-elementti valmiiksi muotoiltu teksti esim. esitettävälle ohjelmakoodille. Koodi näkyy sivuilla kuten html-koodiin on lisätty, hyvä juttu niin. Koodi voidaan merkata koodiksi code elementillä pre-elementin sisällä

Alla open CSS-koodi. Keksi ja koodaa omasi, älä kopioi alla olevaa koodia.

 
  p
   {
	display:block;
	margin-top:1em;
	margin-bottom: 1em;
   }

T30 tee kuvauslista.

Kuvauslistalla voit periaatteessa sisentää tekstiä.

Kuvan voit lisätä img src elementillä, lisää alt teksti kuvaan. Älä kopioi alla olevaa, vaan hae netistä jokin mielenkiintoinen kuva ja lisää sen alle teksti dt, br ja dd:n avulla.
