Ohjeita tehtävien tekoon

MDN docs löydät kaikki HTML elementit. Tee html koodi itse Visual Studio Code ohjelmalla. Lisää metatietoihin sivun tekijän nimi (oma nimesi). Tee tyhjät .css ja .js tiedostot myös.

Tee T0.1-T0.2 tehtävät jsfiddle.net sivujen avulla. Kopioi valmis jsfiddle koodi Visual Studio Code ohjelmaan ja tallenna. Tai käytä vaikka kuvankaappausta valmiin tehtävän kopiointiin. Pääasia on, että Jsfiddlesi on tallessa.

Vie kaikki vastauksesi Teamsiin. Versionhallinnan käyttö tehtävissä on suotavaa.
